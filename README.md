## Tarea : JavaScript - Funciones Flecha - Tradicional

Este proyecto contiene ejemplos de operaciones matemáticas básicas utilizando funciones flecha y tradicionales en JavaScript

Las funciones implementadas son :

- Suma
- Resta
- Multiplicación
- División

## Ejemplo de función tradicional

```javascript
function suma(a,b){
    return a+b
}
```

## Ejemplo de función flecha

```javascript
const sumaflecha=(a,b) => a+b

```

## Cómo ejecutar

1. Descargar el repositorio
2. Abrir el archivo `index.html` en el navegador
3. Abrir la consola del navegador para ver los resultados
